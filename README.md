# Topicbox Dark

A Dark Theme for Topicbox

![Preview of the theme](preview.png)

## Installation

You'll need to:

1. Install Stylus ([Firefox][stylus-ff], [Chrome][stylus-cr])
1. Install [Topicbox Dark][install-github]
